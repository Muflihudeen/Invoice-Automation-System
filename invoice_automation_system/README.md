# Smart Invoice Processing & Financial Document Automation System

## Overview
This project is a Python-based invoice processing system that automates invoice management. It extracts invoice information from PDF files, validates the extracted data, stores it in an SQLite database, and provides reporting and export features.

## Features
- Process invoice PDF files
- Extract invoice information automatically
- Validate invoice data
- Store invoice records in SQLite
- View all invoices
- Search invoices
- Delete invoices
- Export data to CSV
- Export data to Excel
- Generate financial reports
- System logging

## Technologies Used
- Python
- SQLite
- pdfplumber
- pandas
- openpyxl

## Installation

pip install pdfplumber pandas openpyxl

## Run

python main.py
