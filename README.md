# Invoice-Automation-System
a Python-based invoice processing system that automates invoice management
Smart Invoice Processing & Financial Document Automation System built with Python.

Features:
• PDF invoice processing
• Automatic data extraction
• SQLite database
• Invoice validation
• Search & delete invoices
• CSV export
• Excel export
• Financial report generation
• Logging system

Tech Stack:
Python | SQLite | Pandas | OpenPyXL | pdfplumber
